UTO - Unit Testing Online

Olá, bem-vindx ao UTO. Este é um sistema de correção de questões automática para algoritmo. UTO utiliza testes unitários e permite que você teste seus exercícios de maneira fácil e rápida.

Sua primeira vez aqui? Clique aqui !

O que vamos fazer? Iremos executar o seu código, fornecer dados de entrada e comparar os dados de saída com saídas esperadas.

Como pode utiliza o UTO? Acesse as listas de exercícios, crie o seu código e submeta ao UTO.

O seu código pode resultar em Accepted, quando ele está 100% correto.

Ou Failed, quando ele não possui erro sintático, mas não está em conformidade com o que foi solicitado na atividade. Neste caso, ajuste a lógica do seu código, aproveitando os dados dos testes.

Por fim, o seu código pode ter erro sintático Error. Neste caso, use o seu compilador para verificar o erro.

Esta versão do UTO salva progresso localmente (navegador).
