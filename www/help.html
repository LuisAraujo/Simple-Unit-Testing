SUT - Simple Unit Testing

FAQ

Como isso funciona

Iremos executar o seu código, fornecendo dados de entrada e comparar os dados de saída com saídas esperadas.

Como pode utiliza o SUT?

Importe a sua lista de exercicios. Acesse as listas, crie o seu código em qualquer IDE ou Editor de Texto e submeta ao SUT.

O que siginificam as palavras na lista de exercício?

Todas as atividades inicial como a seguinte lable: Pending. Após o envio do seu código, três labels podem aparecer. Quando o seu código está 100% correto, você verá esta lable: Accepted. A seguinte label: Failed, aparecerá quando o seu código não possui erro sintático, mas não está em conformidade com o que foi solicitado na atividade. Neste caso, ajuste a lógica do seu código, aproveitando os dados dos testes. Caso o seu código tenha erro sintático, você verá esta label: Error. Neste caso, use o seu compilador para verificar o erro.

O que significa a porcentagem no banco de dados?

Nó contabilizamos quantos exercicios já forma aceitos, em comparação com o total de exercicios em um banco de dados. Atenção: Ao deletar um banco, todo o progresso é perdido.
